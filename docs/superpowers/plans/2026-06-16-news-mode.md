# News Mode (каркас) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `Topic` entity and generalize the analytics engine from `brand_id` to a brand|topic "scope", so the product has two modes — «Новости» (topic-driven, no brand) and «Бренд» (current).

**Architecture:** New `Topic` table + nullable `topic_id` on `Mention/Incident/Story/Report/Probe` (exactly one of brand_id/topic_id). A small `Scope` abstraction supplies keywords + the owner FK; collection/`update_stories`/`build_daily_digest` take a Scope instead of a bare brand_id (brand path preserved via `scope_for_brand`). New `/news/topics` endpoints; existing analytics endpoints accept `topic_id`. Frontend gets a mode switcher + topic-scoped news workspace.

**Tech Stack:** Python 3.x, FastAPI, SQLAlchemy, SQLite; React (echo-app). Use `python3`. Spec: `docs/superpowers/specs/2026-06-16-news-mode-design.md`. Background/how-to: `docs/HANDOFF.md`.

- [ ] **Step 3:** Create `backend/radar/scope.py`:
```python
from __future__ import annotations
from dataclasses import dataclass, field


@dataclass
class Scope:
    kind: str            # 'brand' | 'topic'
    id: int
    name: str
    keywords: list[str] = field(default_factory=list)
    niche_keywords: list[str] = field(default_factory=list)
    market: str = "ru"
    local_mode: bool = False

    def owner_kwargs(self) -> dict:
        return {f"{self.kind}_id": self.id}


def scope_for_brand(b) -> Scope:
    return Scope("brand", b.id, b.name, b.keywords_list(), b.niche_keywords_list(),
                 getattr(b, "market", "ru"), getattr(b, "local_mode", False))


def scope_for_topic(t) -> Scope:
    return Scope("topic", t.id, t.name, t.keywords_list(), t.niche_keywords_list(),
                 getattr(t, "market", "ru"), False)
```

- [ ] **Step 4: Run** → PASS. **Step 5: Commit** `feat: Scope abstraction (brand|topic owner)`.

        s.add(Mention(topic_id=1, platform="web", post_id=f"p{i}", author="a",

                      created_at=base - timedelta(minutes=i)))
