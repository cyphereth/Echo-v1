# UI Kit — Echo Radar · Приложение (диспетчерский инбокс)

Высокоточная интерактивная реконструкция рабочей поверхности продукта: тёмный
«диспетчерский центр», где PR/SMM-менеджер разбирает пойманные упоминания.

## Запуск
Откройте `index.html`. Файл **самодостаточный**: React + Babel + шрифты грузятся с CDN,
токены и стили встроены инлайн. Кликабельный прототип — выбирайте упоминания, одобряйте
и правьте черновики, переключайтесь на экран зондов.

## Что показывает прототип
- **Инбокс с двумя полосами** — `PR · срочное` (высокая виральность) и `SMM`
  (ответы и возможности). Над лентой — scope-bar со сводкой сигналов в эфире.
- **Карточка упоминания** — платформа, автор, severity-кольцо, спарклайн динамики,
  фаза (растёт/затухает), статус (новое/отправлено/решает человек).
- **Детальная панель** — пост, тон + уверенность + категория, график hot-watch,
  редактируемый черновик ответа, превью Telegram-пуша, действия
  одобрить / править / отклонить (меняют статус, логируют правку, шлют тост).
- **Confidence-гейт** — при уверенности < 0.60 черновик не предлагается, упоминание
  помечается «решает человек» (можно эскалировать в PR).
- **Юмор** — помечен «только вручную», не автопостится.
- **Экран зондов** — AI-онбординг (ссылка на бренд → предложенные ключевые слова и
  хэштеги) + список активных зондов с адаптивным интервалом, площадками и watermark.

## Структура (модульный источник)
| Файл | Содержит |
|---|---|
| `data.jsx` | Демо-данные: бренд «Папа Пицца», упоминания (кейс Маши и др.) |
| `icons.jsx` | `Icon` (Lucide-пути), глифы площадок, логотип `EchoWordmark` |
| `primitives.jsx` | `Button`, `PlatformMark`, `Avatar`, `SeverityRing`, `Sparkline`, `SeverityBadge`, `LaneTag` |
| `shell.jsx` | `Sidebar`, `TopBar` |
| `inbox.jsx` | `MentionRow`, `LaneColumn`, `Inbox`, `StatusFlag` |
| `detail.jsx` | `DetailPanel`, `SnapshotChart`, `TelegramPreview`, `ScoreChip` |
| `probes.jsx` | `ProbesScreen` (AI-онбординг + активные зонды) |
| `app.jsx` | `App` (состояние, экраны, действия) + `ScopeBar` |
| `styles.css` | Стили кита (потребляют токены из корневого `colors_and_type.css`) |

> **Сборка.** Превью-среда не отдаёт локальные суб-ресурсы (CSS/JS по `src`/`href`),
> поэтому `index.html` собран самодостаточным: токены и `styles.css` встроены инлайн,
> а `.jsx` сконкатенированы в один Babel-скрипт (общий scope). Исходники-`.jsx` —
> канонический, читаемый источник; `index.html` пересобирается из них.
