# UI Kit — Echo Radar · Маркетинговый сайт (лендинг)

Высокоточная реконструкция продающего лендинга: тот же тёмный «диспетчерский»
язык, что и в продукте, — спокойный циан как фон, тёплый severity как точка.

## Запуск
Откройте `index.html`. Самодостаточный файл: React + Babel + шрифты с CDN, токены
и стили инлайн. Скролл-страница со смысловыми секциями.

## Секции
1. **Header** — стеклянная шапка: логотип «Echo )))», навигация, CTA «Запросить демо».
2. **Hero** — оффер «Ловите негатив, пока у ролика тысяча просмотров, а не миллион»
   + радар-scope с пойманной карточкой (кейс Маши) и блипами.
3. **Stakes** — кривая виральности: где ловит Echo (на разгоне) против того, где
   замечают обычно (на пике-кризисе).
4. **Pipeline** — конвейер: Сбор → Классификация → Скоринг → Черновик → Доставка.
5. **SeverityDemo** — три состояния severity (залетает / растёт / под контролем).
6. **Platforms** — площадки: IG + TikTok (v1), Twitter + Telegram (скоро).
7. **HumanLoop** — контроль: решение за человеком, confidence-гейт, юмор вручную.
8. **Pricing** — три тарифа по лимиту упоминаний/мес.
9. **CTA + Footer**.

## Структура (модульный источник)
| Файл | Содержит |
|---|---|
| `icons.jsx` | `Icon`, глифы площадок, `EchoWordmark` (общий с app-китом) |
| `primitives.jsx` | `PlatformMark` + карта `PF` |
| `sections1.jsx` | `Header`, `Hero`, `HeroScope`, `MiniMention`, `Stakes`, `Pipeline` + хелперы |
| `sections2.jsx` | `SeverityDemo`, `Platforms`, `HumanLoop`, `Pricing`, `CTABand`, `Footer`, `WsRing` |
| `main.jsx` | `Site` (композиция) + render |
| `styles.css` | Стили лендинга (потребляют токены из корневого `colors_and_type.css`) |

> **Сборка** — как в app-ките: `index.html` собран самодостаточным (токены + `styles.css`
> инлайн, `.jsx` сконкатенированы в один Babel-скрипт). Исходники-`.jsx` — канонический источник.
